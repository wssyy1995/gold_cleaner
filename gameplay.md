# 《金牌保洁升职记》玩法文档

> 基于代码解析整理的项目玩法说明

---

## 一、游戏概述

| 项目 | 说明 |
|------|------|
| **游戏名称** | 金牌保洁升职记 |
| **平台** | 微信小游戏 |
| **类型** | 休闲益智 / 模拟清洁 |
| **核心体验** | 通过选择正确的清洁工具，清理房间内的各种污垢，完成关卡挑战 |
| **画风** | 2D 卡通手绘风格，Canvas 自绘制 UI |

### 1.1 游戏目标
玩家扮演一名保洁员，进入不同的房间场景，利用底部工具槽中的清洁工具，清除画面中所有的污垢（垃圾、污渍、灰尘、霉斑等），在限定时间内完成清洁任务，获得星级评价和金币奖励，解锁更多关卡和工具。

---

## 二、核心玩法循环

```
首页选关 → 加载关卡背景 → 识别污垢 → 选择工具 → 交互清洁 → 全部清洁完成 → 结算评分 → 解锁下一关
```

### 2.1 场景结构
游戏场景分为以下几个核心页面：

| 场景 | 功能 |
|------|------|
| **LoadingScene** | 资源预加载，显示加载进度 |
| **HomeScene** | 首页，展示当前 Stage 的 10 个关卡节点、金币、底部功能入口 |
| **GameplayScene** | 核心游戏场景，进行清洁操作 |
| **ShopScene** | 商店（购买道具/工具） |
| **ToolScene** | 工具背包/管理 |
| **SettingScene** | 设置页面 |

## 三、关卡系统

### 3.1 关卡结构
- 游戏共 **4 个阶段（Stage）**，每个阶段 **10 个关卡**，总计 **40 关**
- 关卡 ID 为全局递增：`Stage1 = 1~10`，`Stage2 = 11~20`，以此类推

| Stage | 名称 | 主题 | 解锁条件 | 特色 |
|-------|------|------|----------|------|
| 1 | 温馨小屋 | 家庭住宅 | 默认解锁 | 基础垃圾、灰尘、水渍 |
| 2 | 公寓大楼 | 公寓 | 通关 Stage1-10 | 新增霉斑、碎纸屑 |
| 3 | 别墅庄园 | 别墅 | 通关 Stage2-10 | 新增锈迹、顽固污渍 |
| 4 | 豪华酒店 | 酒店套房 | 通关 Stage3-10 | 全部污垢类型混合 |

### 3.2 关卡状态
每个关卡在首页节点上显示三种状态：
- **锁定（locked）**：灰色，无法进入
- **已解锁（unlocked）**：彩色边框，可进入挑战
- **已通关（completed）**：显示已获得星级（1~3星）

### 3.3 通关与解锁逻辑
- 首次通关某一关后，自动解锁下一关
- 通关 Stage 的第 10 关后，解锁下一个 Stage 的第 1 关
- 玩家数据和进度保存在本地存储（`wx.getStorageSync`）中

### 3.4 时间限制
每关都有时间限制（60s ~ 150s 不等），倒计时显示在顶部 TopBar。时间到未全部清洁则失败。

## 四、污垢系统

### 4.1 污垢类型一览

游戏中定义了多种污垢类型，每种都有对应的外观和清洁方式：

| 污垢类型 | 名称 | 操作类型 | 清洁配方 | 说明 |
|----------|------|----------|----------|------|
| `paper` / `paper2` / `paper3` | 纸团 | `throw` | `rubbish_bin` | 扔进垃圾桶 |
| `apple` / `banana` | 果皮 | `throw` | `rubbish_bin` | 扔进垃圾桶 |
| `socks` / `socks2` / `socks3` | 脏袜子 | `throw` | `dc_basket` | 放进脏衣篓 |
| `pants` | 脏短裤 | `throw` | `dc_basket` | 放进脏衣篓 |
| `shred_paper` / `leaves` | 碎纸屑/落叶 | `sweep` | `broom` | 扫帚清扫 |
| `dust` / `preset_dust` | 灰尘 | `wipe` | `cloth` | 抹布擦拭 |
| `waterstain` | 水渍 | `wipe` | `mop` | 拖把拖干 |
| `blackmark` | 黑印 | `wipe` | `cloth` | 抹布擦拭 |
| `grime` | 油渍 | `wipe` | `common_spray` → `brush` → `cloth` | 多步清洁 |
| `preset_water_incru` | 水垢 | `spray_wipe` | `common_spray` → `cloth` | 先喷后擦 |
| `preset_urine` | 尿渍 | `wipe` | `cloth` | 抹布擦拭 |
| `preset_liquid` | 液体污渍 | `wipe` | `cloth` | 抹布擦拭 |
| `preset_mildew` | 霉斑 | `wipe` | `cloth` / 专用喷剂 | 抹布擦拭 |
| `mold` | 霉斑（高级） | `wipe` | `spray` → `brush` | 多步清洁 |
| `rust` | 锈迹 | `wipe` | `rust_remover` → `brush` | 多步清洁 |

### 4.2 操作类型详解

#### ① `throw` — 投掷类
- **操作方式**：先点击工具槽选中对应工具（垃圾桶/脏衣篓），工具会弹出到屏幕中央；然后点击对应的垃圾，垃圾会被"扔进"工具中消失
- **动画效果**：垃圾飞入工具、工具摇晃（不倒翁效果）、生气 emoji（选错工具时）
- **代表污垢**：纸团、香蕉皮、苹果、脏袜子、脏短裤

#### ② `wipe` — 擦拭类
- **操作方式**：
  - 方式 A：选中 `cloth`/`mop`/`brush` 等工具后，直接用手指在污垢上滑动
  - 方式 B：点击污垢后自动进入划动模式，工具跳到手指位置跟随滑动
- **进度计算**：根据配置的 `process` 值（默认 3~5 次），每次工具经过污垢区域计 1 次，达到次数后清洁完成
- **视觉反馈**：污垢图片从下往上逐渐消失（裁切效果），进度条显示在污垢上方

#### ③ `sweep` — 清扫类
- **操作方式**：选中 `broom` 后，在 `sweep` 类型污垢上滑动
- **进度计算**：5 次滑动后完成（碎纸屑等）
- **特殊动画**：扫帚拖动时会持续左右翻滚镜像

#### ④ `spray_wipe` — 喷擦类（两阶段）
- **第一阶段（spray）**：使用 `common_spray` 点击污垢 3 次，每次在点击位置生成白色泡沫，并有 5 秒泡沫反应动画
- **第二阶段（wipe）**：泡沫反应后，使用 `cloth` 在污垢上滑动擦拭 3 次
- **等候提示**：第一阶段完成后，污垢上方会出现"灵动岛"风格的胶囊提示，内有清洁图标和滑动进度条动画

### 4.3 预制污垢（Preset Dirt）
部分污垢不是单张图片，而是由算法生成的复杂图案：
- **`preset_dust`**：四边形区域内的灰尘粒子群（如书架上的灰尘）
- **`preset_water_incru`**：水垢斑点群
- **`preset_mildew`**：霉斑簇
- **`preset_liquid`**：液体泼溅效果
- **`preset_urine`**：尿渍晕染效果

这些预制污垢支持配置 `rect`（四边形区域）、`density`（密度）、`count`（数量）等参数。

|--------|------|------|----------|------|
| `cloth` | 抹布 | 🧽 | 1 | 擦拭灰尘、污渍、黑印 |
| `rubbish_bin` | 垃圾桶 | 🗑️ | 1 | 扔纸团、果皮等干垃圾 |
| `dc_basket` | 脏衣篓 | 👖 | 1 | 收脏衣服、袜子 |
| `common_spray` | 万能清洁剂 | 🧴 | 2 | 喷洒清洁剂（第一阶段） |
| `broom` | 扫帚 | 🧹 | 3 | 清扫碎纸屑、落叶 |
| `mop` | 吸水拖把 | - | 2 | 拖干水渍 |

| `vacuum` | 吸尘器 | 🌪️ | 4 | 快速清理大量灰尘 |

### 5.2 高级工具（商店购买）

| 工具ID | 名称 | 价格 | 解锁阶段 | 特殊能力 |
|--------|------|------|----------|----------|
| `magic_sponge` | 魔法海绵 | 500金币 | Stage 2 | 可清洁任何类型污垢 |

### 5.3 工具槽交互
- 底部显示工具槽（类似快捷栏），最多 **8 个槽位**
- 点击工具槽中的工具，工具会以弹性动画"弹出"到屏幕中央偏下位置
- 再次点击弹出的工具或点击空槽位，工具会收回
- 新解锁的工具首次进入关卡时会弹出"工具解锁"弹窗，点击"收取"后自动添加到工具槽第一位

## 六、核心交互详解

### 6.1 页面布局（GameplayScene）

```
┌─────────────────────────┐  ← 9%  顶部区域（浅棕黄色）
│  [进度]  [关卡 3/10]  [⏸] │     TopBar：清洁进度、关卡信息、暂停按钮
├─────────────────────────┤  ← 77% 游戏区域
│                         │
│    [房间背景图]          │     污垢分布在此区域
│    ○ 纸团   ○ 灰尘      │
│         ○ 水渍          │
│                         │
├─────────────────────────┤  ← 14% 底部区域（浅棕黄色）
│  [🧽] [🗑️] [👖] [🧴] ... │     工具槽
└─────────────────────────┘
```

### 6.2 触摸交互流程

1. **选择工具**：点击底部工具槽 → 工具弹出到屏幕中央
2. **清洁操作**：
   - `throw`：点击对应垃圾 → 垃圾飞入工具消失
   - `wipe/sweep`：手指在污垢上滑动 → 每次经过计进度 → 完成消失
   - `spray_wipe`：先点击 3 次喷泡沫 → 再滑动擦拭 3 次
3. **错误反馈**：
   - 选错工具点击污垢：工具左上角跳出 `sad` 表情（😠/😮‍💨），带弹跳动画
   - 轻微震动反馈（支持 haptic）
4. **完成反馈**：
   - 污垢消失，原地迸发"亮晶晶"四角星粒子特效
   - 顶部清洁进度条增加

### 6.3 进度计算
- 每个污垢都有 `process` 字段，决定需要几次有效交互才能完成
- 进度条以蓝色格子形式显示在污垢上方，已完成的格子高亮发光
- 对于图片类污垢，每完成一定比例，图片从下往上裁切消失

## 七、二级背景（隐藏区域）系统

### 7.1 概念
部分关卡中，房间背景图的某些区域藏有"二级背景"（`dirts_deep_area`）。这是游戏中的隐藏探索要素。

### 7.2 发现方式
- 玩家需要在房间背景图上**双击**特定区域（检测半径 80px）
- 首次进入该区域时，会显示一个金色的**放大镜动画**（弹跳呼吸效果）作为提示
- 如果玩家自己发现（非通过金圈提示），会先播放 1.2 秒的放大镜呼吸闪烁动画，再进入隐藏区域

### 7.3 二级背景内的玩法
- 进入后，背景图切换为该区域的特写图（如书架、马桶、洗手台、床铺等）
- 玩家需要清洁完该区域内的所有隐藏污垢
- 全部清洁完成后，**自动淡出回到主关卡**
- 已清洁的二级区域会保存状态，下次进入时不再显示已清完的污垢

### 7.4 提示机制
- **金圈提示**：关卡开始一段时间后，第一个未访问的隐藏区域会显示金色放大镜提示
- **二次提示**：第一关中，如果玩家进入第一个隐藏区后 5 秒还没找到第二个，会弹出 Toast "还有隐藏污垢，再找找！"
- **无效双击反馈**：双击非隐藏区域时，显示 😮‍💨 表情并触发消极振感

### 7.5 示例关卡
- **Stage1-1（老年人卧室）**：书架（preset_dust）、床铺（脏袜子）
- **Stage1-2（卫生间）**：马桶（preset_urine）、洗手台（preset_mildew）

## 八、新手引导系统

### 8.1 触发条件
- 仅第一关触发
- 判断玩家是否已完成过引导（`tutorial_completed` 存储标记）

### 8.2 引导内容（被动式）
引导不会强制中断玩家操作，而是通过视觉提示引导：

| 步骤 | 触发条件 | 引导表现 |
|------|----------|----------|
| 1 | 点击需要垃圾桶的污垢 | 底部 `rubbish_bin` 工具槽闪烁金色边框 |
| 2 | 点击需要脏衣篓的污垢 | 底部 `dc_basket` 工具槽闪烁金色边框 |
| 3 | 点击需要抹布的污垢 | 底部 `cloth` 工具槽闪烁金色边框 |

- 选中正确工具后，目标污垢会显示**白色光圈波纹** + **手指图标**（指尖朝上）提示位置
- `cloth` 类型污垢会显示**Z 字形滑动动画**提示擦拭方式
- 玩家正确使用三种工具各一次后，引导自动结束

### 8.3 全局错误提示
- 在任何关卡中，只要选错工具点击污垢，都会显示 `sad` 表情反馈

## 九、评分与结算

### 9.1 星级评定
通关后根据**剩余时间比例**计算星级：

| 剩余时间比例 | 星级 |
|--------------|------|
| ≥ 60% | ⭐⭐⭐ |
| ≥ 30% | ⭐⭐ |
| < 30% | ⭐ |

### 9.2 金币奖励
```
总金币 = 基础奖励 + 星级加成
星级加成 = (stars - 1) × 25
```
例如：基础 50 金币，3 星则获得 50 + 50 = 100 金币。

### 9.3 结算弹窗
通关后弹出 SettlementDialog，显示：
- 获得的星级（1~3 星）
- 获得的金币数
- 三个按钮：**下一关** / **重玩本关** / **返回首页**

### 9.4 首页通关动画
从关卡返回到首页时，刚通关的关卡节点会播放跳跃变形动画：
- 关卡图标跳动 3 次
- 从"未解锁"状态变形为"已完成"状态，并显示对应星级

## 十、经济系统

### 10.1 金币获取
- 关卡通关奖励
- 星级加成
- 初始赠送 100 金币

### 10.2 金币消耗
- 在 ShopScene 购买高级工具（魔法海绵、电动刷子、除锈剂等）

### 10.3 数据持久化
所有用户数据通过 `DataManager` 保存到本地：
- 金币数量
- 已解锁/已通关关卡
- 每关最高星级
- 拥有的道具和工具
- 设置项（音乐、音效、震动等）

## 十二、特色玩法总结

1. **工具-污垢匹配机制**：不是单一点击，而是需要根据污垢类型选择正确工具
2. **多样化操作**：投掷、擦拭、清扫、喷擦两阶段，不同污垢有不同的交互方式
3. **隐藏探索**：双击背景发现隐藏区域，增加探索乐趣
4. **被动引导**：不强制中断操作，通过金色边框、光圈、Z 字动画温和引导
5. **即时反馈**：丰富的动画、振感、粒子特效，让清洁过程有爽快感
6. **渐进难度**：4 个 Stage 逐步引入新污垢和新工具，保持学习曲线平滑
